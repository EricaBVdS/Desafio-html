O que é HTML

HTML significa Hyper Text Markup Language
É uma linguagem de marcação para criação de paginas da web, com estruturação através de elementos.

(Voltar)

Tags

O elemento HTML é formado pela Tag. Temos a tag de inicio, o conteudo, e a tag de fechamento

(Voltar)

Atributos das Tags

Os atributos são propriedades que uma tag pode ter, os atributos fornecem informações adicionais sobre o elemento
Ex: para imagem usamos a tag img e dentro da tag colocamos o atributo src para especificar o caminho da imagem que será exibida.

Listas

As listas são muito usadas no HTML, existem dois tipos: as listas ordenadas e as nao ordenadas.

As listas ordenadas são definidas pela tag ol e os itens da lista com a tag li.
Para lista não ordenadas usamos a tag ul

Lista ordenada

  1. item 1
  2. item 2
  3. item 3

Lista não ordenada

  • item 1
  • item 2
  • item 3

Links

Os links permitem que usuários naveguem pela página, ou de uma página para outra. A tag a define um link
O atributo mais importante do a elemento é o href, que indica o destino do link.

Isso é um link

Para definir se o link será aberto em outra página usamos o atributo target

Valores do target

_self - Abre o documento na mesma janela/guia em que foi clicado
_blank - Abre o documento em uma nova janela ou guia
_parent - Abre o documento no quadro pai
_top - Abre o documento em todo o corpo da janela

Tambem existe o atributo title que mostra informações como um texto de dica de ferramenta quando o mouse se move sobre o elemento.
